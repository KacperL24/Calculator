# Calculator
My goal for this project is to make a fully functional calculator which can compute any operation. It will be built using Java.

## Functional Requirements
- The calculator must be able to handle multiple operations at once and compute them in the correct order.
- It must support the basic arithmetic operations (+, -, X, /).
- It must not apply any computation until the user gives a signal, either a button or a specific command line.
- The equation given must be valid, an impossible equation (e.g. 42+-2) must return an error to the user.

## Non-Functional Requirements
- The calculator must use a well put together user interface.
- There will be buttons for every usable number and operation available.

## Time
This project will take 3 weeks to make which I believe is a very generous amount of time.
